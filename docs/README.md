## Recommended Obsidian Vault Plugins

Community plugins installed in this vault.


> [!DANGER] Critical
> Enable all HTML elements in Marp Markdown via:
> - Community Plugins -> Marp Slides -> Enable HTML



### Plugins

|Plugin|Version|Author|Description|
|---|---|---|---|
|[Format Hotkeys](https://github.com/anstosa/format-hotkeys-obsidian)|0.1.7|Ansel Santosa|Google Docs style formatting hotkeys|
|[Imgur](https://github.com/gavvvr/obsidian-imgur-plugin)|2.6.3|Kirill Gavrilov|Upload images from clipboard to imgur.com and embed in notes|
|[Marp Slides](https://github.com/samuele-cozzi/obsidian-marp-slides)|0.45.6|Samuele Cozzi|Create markdown-based Marp presentations|
|[Note Refactor](https://github.com/lynchjames/note-refactor-obsidian)|1.8.2|lynchjames|Extract note content into new notes and split notes|
|[Regex Find/Replace](https://github.com/Gru80/obsidian-regex-replace)|1.2.0|Martin Eder|Find and replace text using regular expressions|
|[Toggle Case](https://github.com/MatthewAlner/obsidian-toggle-case)|1.1.0|automattech|Toggle between lowercase, UPPERCASE, and Title Case|

### Installation

All plugins available via **Settings → Community plugins → Browse**.
